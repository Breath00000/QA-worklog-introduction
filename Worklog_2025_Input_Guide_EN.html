Input Guidelines

A detailed guide from A to Z.

⚡ General Rules
📄
1 row = 1 task
Each row represents one task per day. The same member can have multiple rows on the same day.
⏱️
Total Time Spent
Full: 8h  |  Half: 4h  |  Off/Holiday: —
🐛
Bug Report
Can be filled manually or imported from All_Bug_Report via formula / script import.
🔢
Output_Count
Fill in the output count for the task (Bug Report, Bug Checked, Test Case...).
📅
Fill All Weekdays
Fill all Mon→Fri including days off. Run script createInputFromTemplate to pre-generate Date + Member.
⚠️
Column N — Validation
Shows ⚠ if member count is below total active members for that month. Check before submitting.
🗓️ Day Type
☀️
Full
Full working day — 8 hours
Time Spent total = 8
✓ Project ✓ Task ✓ Time ◎ Notes
🌤️
Half
Half working day — 4 hours
Time Spent total = 4
✓ Project ✓ Task ✓ Time ◎ Notes
💤
Off
Day off
Only fill: Date + Member + Day Type
✕ Project ✕ Task ✕ Time ◎ Notes
🎉
Holiday
Public holiday falling on Mon–Fri
Only fill: Date + Member + Day Type
✕ Project ✕ Task ✕ Time ◎ Notes
📊 Column Details
A
No
1, 2, 3...
Sequential integer starting from 1
▶ Run script first
B
Date
dd-mmm-yyyy
Ex: 01-Jan-2025 — 2-digit day, 3-letter English month abbreviation, 4-digit year. Please enter correctly as example for data synchronization.
▶ Run script first
C
Member
Text
Must match REF_MEMBER list
▶ Run script first
D
Project
[NWN2], [TRT]...
Select from REF_PROJECT dropdown. Leave blank when Off/Holiday
◉ Required when Full/Half
E
Task
Regression, Test...
Select from REF_TASK_MAPPING dropdown. Leave blank when Off/Holiday
◉ Required when Full/Half
F
Time Spent
0.5, 1, 1.5, 2...
Integer or x.5 only. Daily total = 8 (Full) or 4 (Half). Leave blank when Off/Holiday
◉ Required when Full/Half
G
Output_Type
Bug Report, Test Case...
Must match REF_TASK_MAPPING. Leave blank when Off/Holiday
◉ Required when Full/Half
H
Output_Count
1, 2, 3... (≥ 1)
Output count for the task. Only fill when Output_Type = Bug Report / Bug Checked.
◉ Only for BR / BC
I
Notes
Free text
Optional. Recommended: fill in holiday name when Day Type = Holiday
◎ Optional
J
Day Type
Full / Half / Off / Holiday
Mandatory dropdown selection. Determines how the day is scored.
★ Mandatory
K
Daily Hours
Decimal, unit: hours
Auto formula, do not fill manually
⚙ Auto formula
L
Target Hours
Decimal, unit: hours
Auto formula, do not fill manually
⚙ Auto formula
M
Status
Text
Auto formula, do not fill manually
⚙ Auto formula
N
Validation
⚠ or blank
Auto formula, do not fill manually. Shows ⚠ if member count for that day is below total active members for the month.
⚙ Auto formula
