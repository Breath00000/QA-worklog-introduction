📋 Worklog — Layer 0

Input Creation Guide

From setting up All_Bug_Report to Refresh All_Input.

00
Set Up All_Bug_Report
Done once when adding a new project
📊 Left Block — Daily Data
A1 = Project  |  B1 = Date  |  C1 onward = member names (one column per member, contiguous — no blank columns in between)

Member names must exactly match REF_MEMBER. Each row = 1 project × 1 date × bug count per member. Data is pulled automatically from project files via IMPORTRANGE.
📋 Right Block — Project List
Fill in: Project Code, Project Name, SpreadsheetId of the corresponding Google Sheet.

Use the formula IMPORTRANGE("SpreadsheetId", "Workload!A1") to pull data from the project file. Each project file must have a sheet Workload with the correct format.

The first time you use IMPORTRANGE, click Allow access to grant access.
💡 Once set up, the left block updates in real time when project files have new bugs — no need to redo this.
↓
01
Create Input Sheet
Run at the start of each month

Set Config!B1 = tháng  |  Config!B2 = năm  →  Click nút ▶ Tạo Input

Script creates sheet Input_xx with all weekdays of the month and active members, pre-filling Date + Member columns.

⚠ Note: If Input_xx already exists, the script will create a new sheet named Input_xx(1), Input_xx(2)... instead of overwriting. Check Config!B3 after creation to select the correct sheet.
↓
02
Refresh Sheet List
Run after creating a new Input sheet

Click nút ▶ Refresh Sheet List

Updates the dropdown at Config!B3 with the list of all Input_* sheets in the file. Select the target sheet before proceeding to the next step.

↓
03
Import Bug Report
Pull bugs from All_Bug_Report into Input

Select target sheet in dropdown Config!B3  →  Click nút ▶ Import Bug Report

Script reads All_Bug_Report and automatically fills Bug Report data into the selected Input sheet:

✅ Existing row (same Date × Member × Project) → updates Output_Count
➕ New row → appends to the bottom
🔢 Duplicate row → removes duplicates, keeps the first occurrence
↓
04
Fill In Remaining Data
Done manually in the Input sheet

Manually fill in the missing columns for each row:

J — Day Type  →  Full / Half / Off / Holiday for each day
D — Project + E — Task + F — Time Spent + G — Output_Type + H — Output_Count  →  for non-Bug Report tasks
💡 Refer to the Input Guidelines for column-by-column filling rules.
↓
05
Check Bug Report
Run to verify data before submitting

Click nút ▶ Check Bug Report

Compares Bug Report data between All_Bug_Report and all Input_* sheets for the year set in Config!B2. Results are written to Config columns J:Z in 3 groups:

🔴 Missing — Exists in All_Bug_Report but not filled in any Input sheet
🟡 Extra — Exists in Input but not found in All_Bug_Report
🟠 Mismatch — Bug count does not match between the two sources
⚠ Note: Script is read-only — it does not modify any data. If errors are found, go back to Step 3 or fix manually.
↓
06
Refresh All Input
Final step — merge everything into All_Input

Click nút ▶ Refresh All Input

Gom tất cả Input_* sheets into All_Input. This is the data source for the entire Layer 1+ pipeline.

💡 Re-run this step whenever changes are made to any Input_* sheet.
